# Astro416-Final-Project
This is the repository for my Astronomy 416 Data science for Astrophysicists course at University of Michigan.
It contains my jupyter notebook code and the dataset used. This project uses a deicision tree to classify asteroids based on their composition. 
This is important because asteroids can give a lot of important information about our early solar system, possibly how water got to earth, and they contain other primodial materials that could help us indiacte how life began. Along with being able to identify how to react to asteroids that might collide with earth.
